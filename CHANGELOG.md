# Changelog

## urban-hermes-v0.2.0

- Published Urban-Hermes as the active Urban Agent runtime.
- Switched package metadata and console entry points to `urban-hermes`.
- Kept `urban-agent` as a compatibility alias that launches the same
  Urban-Hermes runtime.
- Added the CLI-synchronized route workspace frontend for route-tree review,
  node artifacts, worker-review events, and claim-boundary inspection.
- Added route-state, research-memory, and Hermes-runtime validation tests for
  the current release path.
- Removed the legacy `urban_agent` package from the public release surface.

## 0.1.0

- Initial prototype package for the earlier UrbanAgent code path.
