## 指标 1：传统色彩匹配度

- **定义**：建筑立面主色调与当地传统建筑色谱（砖红、青灰、米白、木棕）的偏差程度
- **数据来源**：街景图像。传统色谱暂缺参考库，目前以四种传统色为代理标准
- **计算方法**：提取每栋沿街建筑立面主色调 → 在 CIELAB 色彩空间中计算与四种传统色的最小 ΔE 色差 → 色差越小匹配度越高 → 取所有采样点均值
- **试算要求**：随机选取 5-10 个采样点的街景图像，提取主色调并报告 ΔE 值

## 指标 2：传统材料可见占比

- **定义**：传统建筑材料（砖、石、木、瓦）在建筑立面中的可见面积占比
- **数据来源**：街景图像
- **计算方法**：通过 VLM 或人工标注识别立面中的砖/石/木/瓦区域 → 计算传统材料像素数 / 建筑立面总像素数 → 按采样点汇总
- **试算要求**：选取 5-10 张街景，使用 VLM（图像理解模型）判断立面材料构成

## 指标 3：店招风貌干扰度

- **定义**：商业店招的面积、色彩、位置对历史街区风貌协调性的干扰程度
- **数据来源**：街景图像 + SinoBF-1 建筑功能预测（区分商业建筑）
- **计算方法**：
  1. 从 SinoBF-1 中筛选 AOI 内商业功能建筑
  2. 匹配商业建筑的街景图像
  3. 评估三个维度：店招面积占比、色彩冲突度（店招主色与四种传统色的 ΔE）、位置遮挡度
  4. 综合为 0-1 干扰分：0=无干扰，1=严重干扰
- **试算要求**：筛选 SinoBF-1 中的商业建筑 → 选取前 5 个有街景覆盖的 → 计算干扰分
