# Case Study 1 Task Prompt

你正在帮助完成一项历史街区建成环境分析。研究大方向是：先从原始数据构建历史街区建成环境 X 指标，后续再分析这些 X 对游客历史风貌感知强度 Y 的影响。

## 分析区域

宁波老外滩历史文化街区（AOI: 009_宁波_宁波老外滩_boundary.geojson）

## 可用数据

1. **AOI 边界**：`paper9_heritageIntelligence/data/district_boundaries_v2/district_boundaries/009_宁波_宁波老外滩_boundary.geojson`
   - GeoJSON Polygon，WGS84 坐标
2. **SinoBF-1 建筑功能预测**：`paper9_heritageIntelligence/data/sinobf1/`
   - CSV 格式，包含建筑级别的机器学习预测功能标签（商业/居住/文保/办公等）
3. **OSM 道路与建筑缓存**：`paper9_heritageIntelligence/heritage_district_batch/`
   - 预抓取的 OSM 路网、建筑轮廓、POI 数据

## 任务要求

请完成以下步骤，**不要直接跳到 Y 建模**：

1. **数据可行性检查**
   - 逐一检查上述数据源是否可访问、格式是否正确
   - 验证各数据源的空间覆盖范围是否匹配 AOI 边界
   - 记录哪些数据可用、哪些数据缺失或不可用

2. **X 指标设计**
   - 为该街区设计 3-5 个建成环境 X 指标
   - 将每个指标分类为三类之一：
     - 「可直接计算」— 当前数据足够，可直接执行
     - 「仅可代理」— 当前数据不足以直接计算，但可用接近的代理指标
     - 「暂不可计算」— 当前数据完全无法支持
   - 对可计算和可代理的指标，说明计算思路和方法

3. **中间输出**
   - 对可计算指标，产出中间结果（GIS 图层、表格、图表）
   - 对代理指标，说明代理逻辑和局限性
   - 对不可计算指标，说明缺失哪些数据

4. **纠偏与不确定性标记**
   - 明确标记以下内容：
     - 缺失的数据层
     - 不确定的计算假设
     - 需要人工校验的空间判断
     - 下一轮迭代应记录的经验教训

## 评估关注点

- 输入阶段：问题-数据-方法是否匹配？（Gap 1）
- 推理阶段：空间/时间/人群推理是否可复核？（Gap 2）
- 产出阶段：纠偏钩子是否保留、是否可复用？（Gap 3）
