# Case 2 数据画布准备

把你想让 Urban-Hermes 分析的数据文件放在：

```
D:/UrbanAgents_Case2_Data
```

Urban-Hermes 会自己盘点目录内的文件、字段、空间范围和时间覆盖。数据格式不做限制——真实研究场景下数据格式本来就是多样的。

## 输出目录

```
D:/UrbanAgents_Case2_Output/realistic_dialogue
```

## 说明

- 你有没有观测活力结果变量、时间戳、街景图像或其他数据，由你的研究场景决定。
- 如果某些数据不存在，Urban-Hermes 应当能识别并诚实说明，而不是硬跑一个不成立的模型。这正是本实验要观察的行为。
- 如果你只有建成环境数据、没有活力结果，那也是有效的实验输入。
- 如果你有街景感知分数但没有观测活力，Urban-Hermes 应当把感知指标作为解释变量/情境变量，而不是结果变量。
